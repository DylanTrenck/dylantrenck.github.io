HighlightHub is a passion project I've been working on since it's ideation in November 2024. It is a machine learning model that detects highlight moments in videos through the analysis of audio and temporal attributes as well as reinforced learning. It is a system I've designed to help content creators / streamers find highlight moments in their content in real-time to be used for short-form content creation.
